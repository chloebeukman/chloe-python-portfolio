# 💰 Expense Tracker App

A desktop application built with Python that helps users manage their finances by tracking income, expenses, and financial goals in one place.

---

## 🚀 Features

* Add, update, and delete income and expenses
* Create and manage categories
* Track total income vs expenses
* Calculate and display budget balance
* Set and monitor financial goals
